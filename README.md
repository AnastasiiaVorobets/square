## Technologies
- Node v16.20.2
- NPM
- React
- JS
- SCSS (Flexbox, Transformation)
- BEM

## To run the project
- Make a fork
- Clone the project
- Run `npm install` in your terminal
- Run `npm start`
- Open `http://localhost:3000/`
- (available to deploy application to gh-pages)
